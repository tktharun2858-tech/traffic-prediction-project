# 🚦 Traffic Prediction System

A web application that predicts traffic based on time, day, and location using Machine Learning.

## Features
- Predicts traffic volume
- Shows traffic level (Low, Moderate, Heavy)
- Location-based prediction
- Map integration

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- Machine Learning

## How to Run
pip install -r requirements.txt  
python train_model.py  
python app.py
